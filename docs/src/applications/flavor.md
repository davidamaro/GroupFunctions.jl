# Example: flavor mixing

This page is a placeholder for particle-physics examples that evaluate
`SU(3)` matrix elements and characters in flavor-space calculations.

For now, the most relevant building blocks are:

- [`group_function`](@ref) for individual representation-matrix entries.
- [`character`](@ref) for trace-level expressions.
- [Background on group functions](../background/group_functions.md) for the mathematical setup.

The existing quantum-optics and qubit-transmission examples exercise the same
core APIs with more complete end-to-end calculations.
